# Commerce AI Platform

An enterprise-grade AI commerce platform built with Go that demonstrates modern AI engineering patterns including AI agents, RAG, tool calling, observability, and automated evaluations.

## Features

- AI Shopping Agent
- Product Catalog Search
- Order Management
- Returns Processing
- FAQ Assistant using RAG
- SQLite Backend
- Ollama Integration
- OpenTelemetry Tracing
- Arize Phoenix Observability
- Automated LLM Evaluations

## Tech Stack

- Go
- Gin
- SQLite
- sqlite-vec
- Ollama
- OpenTelemetry
- Arize Phoenix
- React (future)
