# Return Policy

## Return Window

Items can be returned within 30 calendar days from the delivery date.

## Eligible Items

Products must:

- Be unused
- Be unworn
- Include original tags
- Include original packaging

## Non-returnable Items

The following cannot be returned:

- Gift cards
- Final sale items
- Personalized products
- Products damaged by misuse

## Refunds

Refunds are issued to the original payment method after the returned item passes inspection.

Refund processing usually takes 5–7 business days.

## Exchanges

Customers may exchange a product for a different size or color, subject to inventory availability.

## Return Shipping

If the return is due to our mistake, return shipping is free.

Otherwise, customers are responsible for return shipping costs.
